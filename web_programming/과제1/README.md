##21년 웹프로그래밍 과제 1

###과제 내용
html, css, javascript를 활용하여 웹페이지 제작

###과제 조건
* 형식 제한 없음
* html5 웹프로그래밍 입문 교재 1~4장의 내용을 활용
* 배우지 않은 내용은 최소화

###제작한 웹페이지
####주제
귀여운 동물 사진 전시회
####구현한 내용
+ 수업시간에 배운 내용을 최대한 다 담을 수 있도록 함

  + html 
    기본 문법 요소를 최대한 활용
    하이퍼링크 사용
    이미지, 음악, 동영상 첨부
    표, 리스트 제작
    입력 양식을 활용하여 회원가입, 설문지 제작
    
  + css
    홈페이지 배경 색상 적용
    홈페이지 글자 꾸미기
    hover 기능 활용
  
  + javascript
    환영 메시지 띄우기
